### 2. AdSense 광고 슬롯 실제 삽입
현재 AdSense 스크립트(`ca-pub-3050601904412736`)는 `<head>`에만 있고 실제 `<ins>` 슬롯이 없음.  
`AdBanner.tsx` 컴포넌트 생성 후 아래 위치에 삽입:
- `ToolLayout.tsx` 하단 (24개 도구 전체 자동 적용)
- `BlogIndex.tsx` 6번째 카드 이후 인피드
- `Home.tsx` 룰렛 아래, RecommendedPresets 위

### 3. 글 품질 개선 — 완료 ✅
`generatedContent.tsx` 템플릿 전면 개선:
- 8가지 오프닝 변형 (기존 4개)
- **흔한 실수 3가지 섹션** 신규 추가
- **실전 팁 섹션** 신규 추가 (2가지)
- **마무리 한 줄 정리** 신규 추가
- FAQ 답변 4가지 변형으로 내용 충실화
- 실행 순서 4단계 → 5단계로 확장
- 표에 "예외" 행 추가
- 체크리스트 4개 → 5개
- 600개 글 전부 즉시 적용

## 🟠 이번주 (P1)

### UX 개선

#### 모바일 SPIN 버튼 복원 (임팩트 최상)
`SpinButton.tsx`: `hidden md:flex` → `flex`  
모바일에서 SPIN FAB이 현재 완전히 숨겨져 있음.  
Hero 섹션 `pt-24` → `pt-16`으로 줄여 룰렛이 첫 화면에 보이도록 조정.

#### 모바일 ItemEditorModal에 프리셋 추가
데스크탑엔 있는 "빠른 설정" 프리셋이 모바일 모달에 없음.  
`PRESETS` 배열을 공유 파일로 분리하여 두 컴포넌트에서 재사용.

#### 결과 히스토리 패널 (최근 10회)
`use-state-persistence.ts`에 `history: string[]` 추가.  
데스크탑: 편집 패널 하단. 모바일: "기록 보기" 슬라이드업 시트.

### 신기능

#### 타이머 도구 추가 (3~4시간)
"온라인 타이머" 월 검색량 60,000~100,000건 (최상위권).  
`use-sound` 훅 이미 존재. `ToolLayout` 패턴 재사용.  
카운트다운 + 스톱워치 + 뽀모도로 3-in-1.

#### 로또 5게임 동시 생성 (1.5시간)
"로또 5게임 자동 추첨" 검색 의도 정확 매칭.  
현재 1세트만 생성. 게임 수 선택(1~5) + 전체 복사 버튼.

#### ToolsIndex 검색 필터 (2시간)
도구 24개 → 이름+설명으로 검색 + 카테고리 탭(랜덤/텍스트/계산기/날짜).

## 🟡 이번달 (P2)

### 리텐션 핵심 기능

#### 스핀 히스토리 + 통계 대시보드
`spinflow:history` localStorage에 최대 200건 누적.  
항목별 당첨 횟수 바차트 + "가장 많이 나온 항목" 문구.  
데이터가 쌓일수록 떠나기 아까운 심리 → WAU 개선.

#### 점심 룰렛 데일리 미션 (PWA Push)
매일 11시 PWA 푸시 알림.  
당일 첫 스핀 → `spinflow:daily:YYYY-MM-DD`에 저장.  
7일 연속 방문 시 골드 컨페티 + 특수 사운드.  
Duolingo형 스트릭 심리 → 주중 DAU 30~50% 향상 목표.

#### 결과 공유 카드 이미지 (바이럴 핵심)
Canvas API로 "오늘의 점심: 치킨 | spinkorea.kr" 카드 생성.  
Web Share API로 카카오톡/인스타 스토리 직접 공유.  
카드에 URL 워터마크 → 수신자 클릭 유입.

#### 나만의 룰렛 저장소 (최대 10개)
현재 1개 슬롯 → `spinflow:saved:[]` 배열 확장.  
"현재 룰렛 저장" 버튼 + 이름 태그 + 1-클릭 복원.

### 수익화

#### 배달앱 제휴 링크 (즉시 수익)
음식 결과 나올 때 "쿠팡이츠에서 치킨 주문하기" CTA.  
쿠팡이츠 파트너스 CPA: 첫 주문 건당 3,000~5,000원.  
예상: 월 50~100만원 (트래픽 비례 성장).

#### 고임팩트 도구별 전용 SEO 콘텐츠 강화
RPM 높은 도구 순위: BMI > 나이계산기 > 비밀번호 > QR코드 > 로또.  
각 도구 페이지에 블로그 내부 링크 집중 연결.

### 콘텐츠/SEO

#### 신규 랜딩페이지 (TOP 3)
1. `/spin/event` — 이벤트·경품 추첨 전용 (검색량 8,000~15,000)
2. `/spin/penalty` — 벌칙 룰렛 (바이럴 최강, 검색량 5,000~8,000)
3. `/spin/classroom` — 수업용 발표자 뽑기 (교사 커뮤니티 타겟)

#### 블로그 카테고리 인덱스 페이지
`/blog/category/random-draw/` 등 카테고리별 집합 페이지.  
롱테일 집계 키워드 흡수 + 내부 링크 허브 역할.

#### "오늘 뭐 먹지" 구어체 H1 추가 (30분)
`/lunch-menu` 페이지에 "오늘 뭐 먹지" 구어체 H1 삽입.  
월 30,000~50,000 검색량 공략.

## 🟢 다음달 (P3)

### D1 DB 서버 기능

#### 공유 단축 URL (`spinkorea.kr/r/abc12345`)
현재 lz-string URL이 2,000자 초과 경고 발생.  
D1에 룰렛 설정 저장 → 8자리 단축 ID 발급.  
인기 프리셋 TOP 10 페이지 → SEO + 바이럴.

#### 추첨 결과 공증서
스핀 후 "결과 공증하기" → 타임스탬프+항목+결과 서버 저장.  
"이 추첨은 2026-05-16 14:32에 공정하게 진행" 공개 페이지.  
프리미엄 과금 진입점 (PDF 인증서, 로고 삽입).

### 프리미엄 구독 (SpinKorea Pro)
- 무료: 기본 룰렛, URL 공유
- 유료 (월 2,900원): 가중치 룰렛, 커스텀 테마, 룰렛 무제한 저장, 광고 제거
- 예상 6개월 후: 월 구독자 100~300명 → 월 30~90만원

### 고임팩트 신규 도구 (검색량 기준)
| 도구 | 월 검색량 | 난이도 |
|------|-----------|--------|
| 타이머/뽀모도로 | 60,000~100,000 | S |
| 환율 계산기 | 50,000~80,000 | M |
| 숫자야구 게임 | 15,000+ | S |
| 상속세/증여세 계산기 | 30,000~50,000 | M |
| 사주팔자/띠 운세 | 수만 | M |
| 랜덤 닉네임 생성기 | 8,000+ | S |
| 색상 팔레트 생성기 | 6,000+ | M |

### 룰렛 고도화
- **탈락 모드**: 선택된 항목 자동 제거 (게임 체류시간 N배)
- **가중치 룰렛**: 항목별 확률 설정 (프리미엄 차별화)
- **멀티 룰렛**: 2개 동시 스핀 (음식+음료)
- **커스텀 테마**: 크리스마스/무지개/파스텔/다크
